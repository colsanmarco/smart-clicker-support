# Smart Clicker — Support

This repository is for **user support** (issues, troubleshooting, and help requests) for Smart Clicker.

## Contact

- Support email: `smartclicker.support@proton.me`

## Privacy Policy

- Privacy policy: `https://colsanmarco.github.io/smart-clicker-privacy/`

## Report a bug

When reporting a bug, please include:

- Extension version (e.g. `2.4.1`)
- Browser (Chrome/Chromium) version
- OS (Windows/macOS/Linux)
- Steps to reproduce
- Logs from the extension (Options → Logs)
